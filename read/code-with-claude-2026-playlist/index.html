השורה התחתונה

הפלייליסט של Code with Claude London 2026 מציג את המעבר של Anthropic מעולם שבו Claude הוא בעיקר עוזר כתיבה וקוד, לעולם שבו Claude הוא שכבת ביצוע ארגונית. המודל חשוב, אבל הוא כבר לא כל הסיפור. הסיפור האמיתי הוא תשתית: agents שמסוגלים לעבוד לאורך זמן, environments בטוחים, memory, dreaming, MCP, skills, evals, sandboxes, credentials, observability ויכולת לבחור מודל לפי outcome ולא לפי hype.

המסר למפתחים ולסטארטאפים חד: היתרון לא יגיע מממשק יפה מעל מודל כללי בלבד. הוא יגיע ממערכות שמבינות את ההקשר האמיתי של הארגון: הקוד, העיצוב, הכלים, הידע, תהליכי האישור, מדדי ההצלחה והגבולות. ככל שהמודלים מתחזקים, הערך עובר למי שיודע לחבר אותם לעבודה אמיתית בצורה מדידה ובטוחה.

מה מחבר את כל הפלייליסט

  • Agentic infrastructure: Managed Agents, event streams, environments ו-sandboxes הופכים את הסוכן ממודל לתהליך פרודקשן.
  • ידע ארגוני כיתרון: skills, memory ו-MCP הם הדרך להפוך ידע פנימי ליכולת שחוזרת על עצמה.
  • Verification לפני אוטומציה: יותר כוח לסוכן מחייב evals, auditability, permissions ו-review מובנה.
  • מודלים לפי תוצאה: ההחלטה הנכונה אינה benchmark או מחיר לטוקן, אלא cost per successful outcome.
  • עיצוב וקוד מתקרבים: Claude מוצג ככלי שמחבר prompt, design, code, cloud ו-production.
